# Eser Menke — Computer Science Portfolio

This repository hosts my personal computer science portfolio website, built as part of
the B201 Computer Science Lab module at Gisma University of Applied Sciences.

**Live website:** https://esermenke.github.io

## What's in here

The site is a single-page portfolio built with plain HTML and CSS. It is hosted for free
using GitHub Pages.

```
EserMenke.github.io/
├── index.html      # Main page (all sections)
├── style.css       # Styling
├── cv/             # My CV (PDF, written in LaTeX)
├── projects/       # Source files for the exercises shown on the site
├── assets/         # Images and other static files
└── README.md       # This file
```

## Sections of the site

- **About** — a short introduction and why I chose software engineering
- **Skills** — the languages and tools I've started learning
- **Projects** — small exercises I've built while learning
- **CV** — a downloadable PDF version of my CV
- **Contact** — how to reach me

## Tools used

- HTML & CSS for the site structure and styling
- Git & GitHub for version control and hosting (GitHub Pages)
- LaTeX for the CV and the project report

## About me

I'm a first-semester software engineering student. This portfolio reflects where I am
as a learner — the projects are intentionally small and focused on practising the basics.
